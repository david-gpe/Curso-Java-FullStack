# 🚀 Java FullStack – Curso y Prácticas

## 🧠 Descripción

Repositorio que documenta el aprendizaje y práctica de un curso de desarrollo **FullStack con Java**, cubriendo tanto backend como conceptos de frontend.

Incluye ejercicios, ejemplos y configuraciones desarrolladas durante el proceso de aprendizaje.

---

## 🎯 Objetivo

Adquirir y consolidar conocimientos en desarrollo FullStack, incluyendo:

* Backend con Java
* APIs REST
* Manejo de bases de datos
* Integración entre capas

---

## 📚 Contenido del curso

El repositorio incluye prácticas relacionadas con:

### 🟢 Backend

* Java
* Spring Boot
* Creación de APIs REST

---

### 🟡 Base de datos

* SQL
* Modelado de datos
* Consultas

---

### 🔵 Integración

* Consumo de APIs
* Comunicación cliente-servidor
